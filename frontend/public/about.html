Platform Capabilities

Pacify provides comprehensive analysis tools for understanding gun violence patterns and policy effectiveness across the United States. Our platform processes massive datasets using advanced AI and machine learning techniques to deliver insights that inform evidence-based policy decisions.

Interactive Geographic Visualization

County-level choropleth maps with incident heatmaps, real-time cursor tracking, and state-specific context panels. Zoom from national trends to local neighborhood patterns with precision geocoding.

AI-Powered Policy Analysis

Google Gemini 2.5 Pro analyzes 1,964+ policy records across 20+ categories, generating human-readable explanations and impact assessments for every state policy change since 2019.

Temporal Data Navigation

Timeline controls enable year-by-year analysis with smooth transitions, auto-play functionality, and correlation analysis between policy changes and violence trends.

Real-Time News Integration

MediaCloud API integration provides current gun violence news coverage across all states, updated daily with sentiment analysis and geographic correlation.

Advanced Analytics Dashboard

Statistical analysis tools with trend detection, correlation matrices, and predictive modeling capabilities for researchers and policymakers.

Mobile-Optimized Interface

Responsive design ensures full functionality across all devices, with touch-optimized controls and adaptive layouts for field research.

Data Quality Assurance

Every data point undergoes rigorous validation: coordinate verification, duplicate removal, format standardization, and cross-source verification. We maintain 99.7% accuracy in geocoding and 100% policy record completeness.

Built for Johns Hopkins Excellence

We apply Wall Street-level quantitative analysis to public policy challenges. Our methodology combines academic rigor with practical application, using the same advanced techniques that predict market movements to understand violence patterns.

Platform Limitations

Transparency requires acknowledging limitations: We cannot predict individual incidents, prove causation (only correlation), account for unreported incidents, show immediate policy effects, or include all private transactions. Our platform provides powerful trend analysis tools, not predictive capabilities.
