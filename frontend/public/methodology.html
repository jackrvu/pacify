methodology

the science behind pacify
rigorous data science for gun violence analysis
pacify processes 270k+ gun violence records through ai-powered policy analysis and statistical modeling to provide actionable insights for evidence-based policy decisions.

data processing pipeline

our platform transforms raw data into actionable insights through a streamlined six-step process:

1

data collection

automated ingestion from gun violence archive, rand corporation, and state databases with validation.

2

geocoding

precise coordinate mapping using google geocoding api for accurate geographic visualization.

3

data cleaning

duplicate removal, format standardization, and quality validation across all sources.

4

ai analysis

google gemini 2.5 pro analyzes 16,944+ policies with constitutional interpretation.

5

statistical modeling

arima time series analysis, correlation matrices, and significance testing.

6

visualization

interactive maps, charts, and dashboards using leaflet.js and d3.js.

technical stack

built with modern web technologies for performance and scalability:

frontend

  • react 18 with leaflet.js mapping
  • d3.js for data visualization
  • webgl-accelerated rendering

backend

  • python with pandas & geopandas
  • google gemini 2.5 pro api
  • automated data validation

limitations

we identify correlations, not causation. unreported incidents are not included. policy effects may take years to manifest. we provide information for decision-makers, not predictions.
