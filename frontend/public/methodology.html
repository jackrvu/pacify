Methodology

Abstract

This document outlines the methodology used in the Pacify gun violence data visualization platform. Our approach combines multiple data sources, statistical analysis, and interactive visualization techniques to provide comprehensive insights into gun violence patterns across the United States.

Data Sources

Gun Violence Archive (GVA)

Primary source for mass shooting incidents, providing detailed information on location, casualties, and circumstances of each event.

State Policy Database

Comprehensive database of state-level gun control policies and legislation changes over time.

Demographic Data

U.S. Census Bureau data for population demographics and geographic information.

Data Processing

Data Cleaning and Validation

All datasets undergo rigorous cleaning processes including:

  • Removal of duplicate entries
  • Standardization of geographic coordinates
  • Validation of date formats and ranges
  • Cross-referencing with multiple sources for accuracy

Geographic Mapping

Incident locations are geocoded and mapped using:

  • Leaflet.js for interactive mapping
  • County-level choropleth visualization
  • Heat map clustering for density analysis
  • State-level policy correlation mapping

Statistical Analysis

Our analysis includes:

  • Time series analysis of incident trends
  • Geographic clustering analysis
  • Policy impact correlation studies
  • Demographic pattern identification

Visualization Techniques

Interactive Timeline

Dynamic timeline controls allow users to explore data across different time periods, with smooth transitions and real-time updates.

Multi-layer Mapping

Layered visualization approach combining county choropleth, incident heatmaps, and policy markers for comprehensive analysis.

Limitations

This analysis is subject to several limitations:

  • Data availability and reporting inconsistencies across states
  • Potential underreporting of certain types of incidents
  • Geographic accuracy limitations in rural areas
  • Policy classification challenges and interpretation variations

Technical Implementation

Frontend Architecture

Built using React.js with Leaflet for mapping, featuring responsive design and real-time data updates.

Data Processing Pipeline

Automated data processing pipeline using Python for data cleaning, analysis, and JSON generation for frontend consumption.
